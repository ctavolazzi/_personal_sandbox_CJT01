## Design Pattern
Reuse existing pattern from `gemini_client.py`:
```python
class OpenAIClient:
    COMPONENT_NAME = "openai"  # Used for config.get_mode()

    def generate(self, prompt: str) -> dict:
        if api_config.is_mock(self.COMPONENT_NAME):
            return self._load_fixture(prompt)
        else:
            return self._call_live_api(prompt)
```
