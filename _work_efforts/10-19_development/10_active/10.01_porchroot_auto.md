## Objective
Build a "high-value" social media content factory that:
1. Generates branded quote images using Python/Pillow (reliable text rendering)
2. Generates captions + hashtags using existing GeminiClient (no API key duplication)
3. Tracks quote history to avoid repetition

## Decision Made
**Via Decision Matrix Analysis:**
- Winner: "Reuse existing GeminiClient" (100% score)
- Rationale: Best DRY compliance, fastest to implement, leverages existing infrastructure
- Trade-off: Slightly less isolated than standalone build

## Architecture
```
projects/porchroot-auto/
├── factory.py          # Main script (imports GeminiClient from api-testing-framework)
├── quotes_db.json      # Quote database
├── used_quotes.txt     # History tracking
├── assets/
│   ├── fonts/          # .ttf files
│   └── backgrounds/    # Dark texture images
└── output/             # Generated posts (.jpg + .txt caption files)
```
